軟體設計工程師

Software Design Engineer
  • 設計AOI系統軟體架構
  • 導入開源專案
  • 測試公司軟體架構穩定度

前端工程師

Front-End Engineer
  • 熟悉網頁版面規劃、套版、特效互動設計
  • 與UI/UX Designers及後端工程師協作開發Web系統介面
  • 熟悉前端建置與佈署，與後端 API 進行串接
  • 負責網站前端開發，優化用戶端功能設計
  • 確保網站支援不同裝置，例如桌面版、手機版等
  • 確保UI / UX設計的技術可行性

後端工程師

Back-End Engineer
  • 開發、維護並優化後端功能與伺服器架構
  • 和App團隊以及前端團隊一起設計服務框架
  • 支援其他專案開發及規劃
  • 協助新的技術、工具或應用的研究與導入評估

SRE/DevOps

SRE/DevOps
  • 負責架構規劃與建置雲端環境，專案技術支援，協助維運與異常問題排除。
  • 熟悉Linux與Windows的服務建置，撰寫自動化腳本程式取代複雜重複性高的流程工作。
  • 根據研發使用需求不同，規劃與需求高度相依性的不同環境(ex. 開發環境, 測試環境, 營運環境)

BIOS工程師

BIOS Engineer
  • 從事電腦BIOS與BIOS相關應用軟體開發、除錯、維護
  • BIOS應用規格及流程撰寫
  • 熟C語言，熟組合語言尤佳

產品經理

Product Manager
  • 負責所有產品相關的工作
  • 調研市場和競品，及時收取分析行業信息
  • 規劃產品發展方向，給出合理的路徑
  • 策劃、分析業務流程，製作產品原型
  • 與技術研發團隊對接，跟進開發中的每一環節
  • 負責最終驗收並跟進上線，通過反饋持續改進產品
  • 負責相關產品設計
